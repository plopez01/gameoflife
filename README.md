# gameoflife
Conway's Game of Life coded in Processing!

## Controls
| Action                  | Control           |
|-------------------------|-------------------|
| Create cells            | Mouse left click  |
| Remove cells            | Mouse right click |
| Start/Stop game         | Spacebar          |
| Clean game              | C key             |
| Initialize random cells | R key             |
| Sim speed and cell size | Modify code ;)    |
